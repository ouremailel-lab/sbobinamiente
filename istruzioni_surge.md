# Come installare Surge

Apri il terminale e digita:

npm install --global surge

# Questo comando installerà Surge in modo che tu possa usarlo ovunque dal terminale.

# Se hai ricevuto un errore di permessi, usa questo comando nel terminale:

sudo npm install --global surge

# Ti verrà chiesta la password del Mac. Dopo l’installazione, potrai usare Surge normalmente.
