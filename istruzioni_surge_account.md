# Come creare un account Surge

1. Apri il terminale nella cartella del tuo sito.
2. Digita il comando per pubblicare il sito (sostituisci con il tuo dominio):
   ```
   surge . www.sbobinamente.it
   ```
3. Surge ti chiederà di inserire la tua email.
   - Se non hai già un account, inserisci la tua email e scegli una password quando richiesto.
   - Se hai già un account, inserisci email e password per accedere.
4. Dopo la registrazione o il login, Surge pubblicherà il sito sul dominio indicato.

**Nota:**  
Se non hai ancora configurato i DNS su IONOS, segui le istruzioni per puntare il dominio a Surge.
